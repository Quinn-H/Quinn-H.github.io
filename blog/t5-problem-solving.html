Problem Solving

   

Tell your non-tech friend a story about a time you: Got blocked on a simple problem

I was stuck by question #3 in eloquent JavaScript book. This assignment required us to write JavaScript code to create a string that represents an 8x8 grid. At first, I know I need to use a ‘for loop’ inside another ‘for loop’. But I didn’t know how to write an if/else statement to put a ‘space’ on the first character of an odd line and a ‘#’ on the first character of even line. It took me 2 hours to figure out how. I have applied to every problem solving technique I learned to this problem, but I still can’t solve it until I found a video on YouTube that teach how to solve this problem. I found solving a problem by looking at the solution is a double edged sword. It can hurt you if you don’t try to solve the problem by yourself at first. When you need to see a solution, you need to ensure you understand the solution very well and you can solve similar problem next time.

    Explain to your non-tech friend

  • .map()

    • The map() method will make a new array and it will calling the provided function for every array element 

                      var num = [4, 9, 16];
var roots = num.map(Math.sqrt);
// roots is now [2, 3, 4]
// num is still [4, 9, 16]
  • .filter()

    • The filter() method creates a new array with all elements that pass the test by the provided function. 

                      function isSmaller(value) {
return value <= 30;
}

var filtered = [12, 5, 8, 130, 44].filter(isSmaller);
// filtered is [12, 5, 8,]
  • .reduce()

    • The reduce() method reduces the array to a single value. 

                      var numbers = [1, 2, 3, 4];

function getSum(total, num) {
    return total + num;
}
function myFunction(item) {
    document.getElementById("demo").innerHTML = numbers.reduce(getSum);
}
