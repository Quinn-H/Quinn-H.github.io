What is grid based design?

Grid system is a structure comprising a series of different size boxes. Using grid system would provide a way that developer can work with to adjust the structure and content of webpage. Grid based design can a beneficial method to design webpage in a manageable and readable way.

What is all the hype about responsive webpages?

Responsive webpage will provide a user-friendly view experience from desktop to smartphone. The content of webpage will resize automatically when user change size of the Brower.

What is semantic structure?

Semantic structure makes HTML Tags have actually meaning to both developer and computer. For example, <title>, <article> and <table> tag will explain what is the purpose of the text in its group.
