JavaScript Basic

   

How does JavaScript compare to HTML and CSS?

HTML is markup language which used to create structuring content. CSS used for visual styles. JavaScript is a programming language which used for creating website interactive functions.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

A control flow example from everyday life: If today is sunny, then walking my dog in the Park, else tell my dog poop on the training pad. A loop example from everyday life: Telling my puppy stop poop on the carpet everyday until my puppy not poop on the carpet.

Explain the difference between accessing data from arrays and object literals.

When we access data from array, we need to refer the index number of data in the array. Object has key values and property values; you need to refer the key value to access the property values.

Explain what functions are and why they are useful.

Function contains a group of code that you can repeat use them. It will eliminate code duplication and save time in writing code.
