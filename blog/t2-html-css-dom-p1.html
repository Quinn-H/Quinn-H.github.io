Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

As I understand it, if you think about building a house, first you need the brick or wood to build the foundation of the house. HTML would be the tool that to build the structure of housing. CSS would be the decorator to make the house looks beautiful and nicer. DOM is your future renovation manual; it tells you how each component work in the house and what you do if you want to make a change of the house.

What is meant by boxifying design?

Boxifying design is a concept to help you understand the structure of the webpage. Every element you see in a webpage; it is in a box.

What is the box model?

The box model is the concept of the design and layout of webpage. Every HTML element is wrapped in a box. It consists of: margins, borders, padding, and content.c
