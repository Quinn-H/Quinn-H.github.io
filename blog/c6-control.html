Control

When programming, what are some of your triggers? What causes a disproportionate emotional response?

I found my triggers during minesweeper assignment. When I work on reset game function for minesweeper I was being frustrated because I can’t make reset function work. I was struggle hours on it without any progress. I finally give up on it.

How did you find the Siberian North Railroad exercise (Stop, Breathe, Notice, Reflect, Respond)?

The Siberian North Railroad exercise help me find alternative solution for reset function. After I got stuck by reset function for hours, I felt really tired. Then I laid on couch for some rest and think about why reset function not working. Suddenly, an idea come up, I can use browner refresh function as alternative solution for reset button because they do same job on this satiation.
